Этот проект содержит два задания по многопоточному программированию на C++:

Программа автоматически протестирует работу с разным количеством потоков (1, 2, 4, 7, 8, 16, 20, 40) и выведет таблицу с временем выполнения и ускорением.

```
2	5.3421	1.97
4	2.7834	3.79
...
```

#### Задание 2: Клиент-серверное приложение

Для запуска с количеством задач по умолчанию (100 на клиента):

```bash

```

Для запуска с自定义нным количеством задач (например, 500):


Программа создаст три файла с результатами:
- `client_sin_results.txt` - результаты вычисления синуса
- `client_sqrt_results.txt` - результаты вычисления квадратного корня
- `client_pow_results.txt` - результаты возведения в степень

После выполнения программа автоматически запустит тесты для проверки результатов.

## Дополнительные возможности

### Привязка потоков к ядрам (опционально)

Для привязки потоков к конкретным ядрам можно использовать `pthread_setaffinity_np`:

```cpp
#include <sched.h>
#include <pthread.h>

void set_affinity(std::thread& t, int core_id) {
    cpu_set_t cpuset;
    CPU_ZERO(&cpuset);
    CPU_SET(core_id, &cpuset);
    
    pthread_t thread = t.native_handle();
    pthread_setaffinity_np(thread, sizeof(cpu_set_t), &cpuset);
}
```

Сравните результаты с привязкой и без привязки к ядрам.
